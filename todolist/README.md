# TodoList备忘录

#### 项目介绍
* 内容：一个简单的任务备忘录，可添加待办任务，删除已办任务，任务顺序重排等
* 技术：react框架，redux数据管理模式
* 备注：此项目为本人react学习项目，部分代码参考自其它react开源项目
